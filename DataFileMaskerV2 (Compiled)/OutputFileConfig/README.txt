IBM_TYPES.ini and Extensions.ini
________________________________

These two files allow the user to add any ibm type they wish as long 
as they have knowledge of the corresponding extension. 

To use these files simply match the IBM type name with the corresponding extension


EX. 

IBM_TYPES.ini 
RICHTEXT
WORD




EXTENSIONS.ini
rtf
doc






--------------------
Make sure there are no spaces as well make sure they corresponing extensions is on 
the same line
